About

Stride is an anime statistics site that collects MyAnimeList (MAL) data to perform tasks such as tracking over time and both item-based and user-based recommender systems. It is not in any way meant to be an alternative or a replacement to MAL.

Statistics

Overview

Stride tracks the MAL statistics of the top 2000 MAL shows (by popularity) over time, updated daily. Additionally, it also calculates the mean and variance of user show ratings, updated weekly. The user data was taken from RedditAnimeList.

Item-based recommender (beta)

Overview

Based on the statistics data from the user show ratings for the top 500 shows, Stride performs item-based collaborative filtering to determine the similarity between shows. For each given show, the six most similar shows are displayed as recommendations in the show's statistics page.

The algorithm used is a modified version of Adjusted Cosine Similarity; extra checks are performed to filter out shows that are exceptionally high rated or popular, as these tend to me recommended quite frequently. It is hoped that lesser known shows can be discovered this way.

User-based recommender (beta)

Overview

For the user-based recommender, a user enters their MAL profile then stride takes a random sample of 1000 RedditAnimeList users and finds the few most similar to the given MAL. (Indeed, there is an element of randomness to this. I decided this would make things interesting.) Recommendations are based off what the similar users liked that the given user has not yet watched.

The algorithm used is a modified version of k-nearest neighbors (kNN). First, cosine similarity is used to determine the users in the random sample that are most similar. Then a rating map scheme is constructed between the most similar users and the given user (to adjust for the fact that people tend to use the 1-10 scale subjectively). The highest of the average of the shows in the rating map are selected and displayed to the recommender.
